# chs-kaaiyun-journal

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存放，可通过 GitHub Pages 直接访问。仓库不针对特定域名或网站，仅作为通用页面存放空间。

## 目录说明

```
├── docs/               # 存放所有 HTML 页面（根目录）
│   ├── index.html      # 默认入口页
│   └── ...
├── README.md           # 本文件
```

所有页面均放置在 `docs/` 目录中，便于启用 GitHub Pages 时直接以根目录形式访问。如需新增页面，直接在 `docs/` 下添加 `.html` 文件即可。

## 页面归档说明

- 每个 HTML 文件均为独立页面，内容可包含文本、图片、样式、脚本等。
- 页面间无强制关联，可单独访问。
- 文件名建议使用简短、有意义的英文或拼音，避免特殊字符。
- 若需引用外部资源（如样式库、字体），建议使用 CDN 或相对路径。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 不保证所有页面长期可用或内容不变。
- 如需提交页面，请通过 Pull Request 或联系维护者。
- 不建议直接使用本仓库用于生产环境，仅作归档展示用途。
